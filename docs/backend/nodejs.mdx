# Node.js

## Overview

Node.js is the most widely used JavaScript runtime, built on Chrome's V8 engine.

It allows for server-side scripting and has a vast ecosystem of libraries and tools.


## Installation (macOS)

Install using [pnpm](./nodejs/pkg-mgm/pnpm/nodejs.mdx). This allow you to install multiple versions of Node.js.
