# Backup & Restore

## Backup

This command backs up a MongoDB database (collections, indexes, ...) to a specified directory `/backup/{database}`:

```bash
mongodump --uri="mongodb+srv://{username}:{password}@{host}/{database}" --out=/backup/ --gzip
```


## Restore

This command restores a MongoDB database from a specified directory `/backup/{database}`:

```bash
mongorestore --uri="mongodb+srv://{username}:{password}@{host}/{database}" --gzip /backup/{database}
```

## Import
```sh
mongoimport --jsonArray --host localhost:27017 --db test --collection restaurants --file primer-dataset.json
```

## Export
```sh
mongoexport --jsonArray --pretty --host localhost:27017 --db test -c restaurants --out newdbexport.json
```
