# Linter

Integration with Linter


## Installation

<PackageManagerTabs command="install --save-dev @vitest/eslint-plugin" />

Update ESLint config file:

```js title="eslint.config.js" {2,8}
import { defineConfig } from "eslint/config";
import vitest from '@vitest/eslint-plugin'

export default defineConfig([
  {
    name: 'Vitest',    
    files: ['**/*.test.ts(x)?'],
    extends: [
      vitest.configs.recommended,
    ],
  },
]);
```
