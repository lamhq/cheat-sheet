# React

Install and configure ESLint for a React (TypeScript) project.


## Installation

Finish the [basic setup](../installation.mdx#basic-setup) first, then run below command to install additional packages:

<PackageManagerTabs command="install --save-dev eslint-plugin-react eslint-plugin-react-hooks globals" />

Update ESLint config file:

```js title="eslint.config.js" {2,3,10,16,17}
import { defineConfig } from "eslint/config";
import reactPlugin from 'eslint-plugin-react';
import reactHooks from 'eslint-plugin-react-hooks';

export default defineConfig([
  {
    name: 'TypeScript React',
    files: ['**/*.tsx'],
    languageOptions: {
      globals: globals.browser,
    },
    settings: {
      react: { version: 'detect' },
    },
    extends: [
      reactPlugin.configs.flat.recommended,
      reactHooks.configs['recommended-latest'],
    ],
  },
]);
```
- The shared configs do not preconfigure `languageOptions.globals`. For most of the cases, you probably want to configure some properties by yourself.
- `react: { version: 'detect' }`: automatically detect the React version from your dependencies


## Integrate with Bundler

<PackageManagerTabs command="install --save-dev eslint-plugin-react-refresh" />

If you're using Vite:

```js title="eslint.config.js" {2,8}
import { defineConfig } from "eslint/config";
import reactRefresh from "eslint-plugin-react-refresh";

export default defineConfig([
  {
    files: ['**/*.tsx'],
    extends: [
      reactRefresh.configs.vite,
    ],
  },
]);
```
