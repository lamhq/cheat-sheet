# Object

## Defining object

```js
const newUser = {
  id,
  username,
  email,
  foo (a, b) { ... },
}
```

From key-value pairs (ES10):
```js
const superHeroArray = [['name', 'Bruce'], ['age', 32]];
const superHeroObject = Object.fromEntries(superHeroArray);

console.log(superHeroObject); // { name: 'Bruce', age: 32 }
```


## Extracting object properties

Object Destructuring (ES6):

```js
const response = { id: 1, username: "Batman", email: "bruce@gmail.com" };
const { id, username, email } = response;
console.log(id, username, email); // 1 Alice a@b.com
```

Extracting multiple properties using rest operator (ES6):

```js
const { name, ...otherDetails } = { name: "Albert", age: 25, city: "Bangkok City" };
console.log(name); // Output: Albert
console.log(otherDetails); // Output: { age: 25, city: "Bangkok City" }
```


## Merging objects

Use spread operator (ES6):

```js
const user = { name: "Albert", age: 25 };
const updatedUser = { ...user, city: "Bangkok City" };
console.log(updatedUser); // Output: { name: "Albert", age: 25, city: "Bangkok City" }
```


## Get field's values

```js
const superhero = { id: 9089, name: 'Batman', age: 32 };
console.log(Object.values(superhero)); // [9089, 'Batman', 32]
```

## Get key-value pairs

```js
const superhero = { id: 9089, name: 'Batman', age: 32 };
console.log(Object.entries(superhero)); // [ [ 'id', 9089 ], [ 'name', 'Batman' ], [ 'age', 32 ] ]
```
