# Learning

## Summarize Web Page

When reading a long web page, you can use an AI agent to quickly summarize the key points for you. Just send it the URL of the web page, and it can generate a concise summary.

<Prompt
  title="Summarize a Web Page"
  description="Send this prompt to your AI agent. Then paste the URL of the web page you want to summarize."
  prompt={summarizeWebPage}
/>


## Take note from video courses

When watching a video course, you can use an AI agent to take notes for you. Just send it the transcript of the video, and it can generate a summary or key takeaways for you.

<Prompt
  title="Summarize a Video Lecture"
  description="Send this prompt to your AI agent. Then paste the transcript of the video lecture you want to summarize."
  prompt="Give me the gist of the lecture base on the provided transcript"
/>
