# Retrievers

## Introduction

A retriever is an interface that returns documents given an unstructured query. It doesn't store documents, just retrieve them.

Retrievers accept a string query as input and return a list of `Document` objects as output.

```python
retriever.invoke("query") → [Document, Document, ...]
```


## Vector Store

| Aspect | Vector Store | Retriever |
|---|---|---|
| **Purpose** | Store & search vectors | Return documents from query |
| **Input** | Documents + embeddings | Unstructured query string |
| **Output** | Similarity scores or docs | List of Document objects |
| **Conversion** | All vector stores convert to retrievers | Broader interface |

All vector stores can be cast as retrievers, but not all retrievers use vector stores (e.g., Wikipedia, Arxiv, Elasticsearch).
