# Zero-Shot Prompting

## Introduction

**Zero-shot prompting** is a type of prompt where the model generates output for a task it has **not been explicitly trained on**. The model performs the task using its pre-existing knowledge rather than specific training examples.

Example: "Create a list of the 10 must-visit cities in the world in no particular order."

Zero-Shot is popular because it is:
- Most intuitive approach for beginners
- Simple and straightforward—just ask questions
- Requires minimal setup or preparation

## Key Characteristics

- No specific training data provided for the task
- No examples given to guide the model
- Relies entirely on the model's learned knowledge
- The model infers how to complete the task from the prompt alone

## Limitations

- **Accuracy concerns**: Output may not match exactly what you're looking for
- **Limited scope**: Guidance and context are minimal
- **Less control**: Relies solely on the model's pre-existing knowledge
- **Not fine-tuned**: Cannot be customized for specific use cases
