# AI Services

## Claude (Anthropic)

### Pros
- Best writing quality of any AI
- Lower hallucination rate
- 200K to 1M token context window
- Extended thinking for hard problems
- Skills, Connectors, Cowork agent
- Doesn't train on your conversations

### Cons
- Can't generate images
- Message limits feel tight on Pro
- Sometimes too cautious / over-refuses
- Smaller plugin ecosystem

### Pricing
- Free: Sonnet, limited msgs/day
- Paid: Pro $20/mo, Max $100+/mo

### Best For
Writers, analysts, researchers, coders

### When to use
- You need natural-sounding writing
- Analyzing long docs or datasets
- You want honest, careful answers
- You need AI to do work (Cowork)

**Heads up:** No images. Can be overly cautious.


## ChatGPT (OpenAI)

### Pros
- Most versatile all-around AI
- Image + video generation built in
- Huge plugin ecosystem (GPT Store)
- Voice mode is genuinely impressive
- Strong at coding and tech tasks
- Biggest community and resources

### Cons
- Writing often sounds "AI-generated"
- Can be confidently wrong
- Free tier getting more restrictive
- Trains on your conversations by default

### Pricing
- Free: GPT-4o mini, limited GPT-5
- Paid: Plus $20/mo, Pro $200/mo

### Best For
Generalists, creatives, developers

### When to use
- You need images or video created
- You want to talk to AI by voice
- You need third-party plugins
- Creative brainstorming sessions

**Heads up:** Always double-check facts. Writing needs editing.


## Gemini (Google)

### Pros
- Deep Google Workspace integration
- Biggest context window (up to 2M)
- Strong multimodal capabilities
- Real-time info via Google Search
- YouTube and Maps data built in
- Very generous free tier

### Cons
- Writing quality can feel flat
- Less reliable for nuanced reasoning
- Best value is locked to Google apps
- Still maturing as a product

### Pricing
- Free: Gemini Pro + search, generous
- Paid: Advanced $20/mo (Google One AI)

### Best For
Google Workspace power users

### When to use
- You live in Gmail, Docs, Drive, Sheets
- You need real-time current info
- Working with very long documents
- You need YouTube or Maps data

**Heads up:** Great inside Google. Less useful outside it.


## Copilot (Microsoft)

### Pros
- Built into Word, Excel, PowerPoint
- Edits and creates Office docs live
- Powered by GPT (strong foundation)
- Enterprise security out of the box
- Always available in Windows
- Free tier is surprisingly useful

### Cons
- Best stuff needs M365 license
- Feels limited outside Microsoft apps
- Less flexible than Claude or ChatGPT
- Can surface sensitive docs if misconfigured

### Pricing
- Free: Chat + Bing search + image gen
- Paid: Pro $20/mo, M365 $30/user/mo

### Best For
Teams running on Microsoft 365

### When to use
- You work in Word/Excel/PPT/Outlook
- Your company pays for M365 already
- You need AI inside existing workflow
- Enterprise compliance is required

**Heads up:** Needs proper permissions setup. Weak outside Office.
