## Extended Thinking

Extended thinking is a feature that pauses Claude before responding, allowing it to work through problems step-by-step internally before providing the final answer.

Extended thinking provides a formal, built-in way to request deeper analysis to get better results.

**How It Works:**
- Claude reasons through the problem
- Considers different angles and perspectives
- Checks its own logic
- Delivers a more careful, deliberate answer

**When to Use Extended Thinking:**
- Complex tasks (document analysis, PowerPoint creation): Significant difference between responses with and without extended thinking
- Intricate problems that benefit from multi-angle analysis


## Powerful Research

The optimal configuration for conducting the most powerful research using Claude.

**Three Essential Elements:**
1. **Enable both Web Search AND Research** – Don't choose just one
2. **Use the highest-tier model you have access to** – For Pro plan users, this is **Opus**; free users would use **Sonnet**
3. **Activate Extended Thinking** – This enhances analysis beyond the base model capability

**The Result:**
When you combine web search + research + highest model + extended thinking, you get Claude's most powerful research capability.

**Caveats:**
- Research feature takes a **long time** to execute
- Uses **significant portions** of your monthly usage allowance
- A single research query with Opus and extended thinking can consume most of your session's allowance

**When to Use This Combination:**
Only deploy this powerful trio when producing high-stakes deliverables:
- Competitive analysis
- Leadership-level reports
- Critical business decisions
- Other crucial work requiring absolute best results

**Alternatives**: Use **Web Search + Sonnet + Extended Thinking** (lighter usage). This is often sufficient for most non-critical tasks


## Projects

Projects are self-contained workspaces that help organize your Claude work by creating separate chat histories and knowledge bases, preventing the mess of having many disconnected conversations.

### Key Features
- Each project has its own chat history
- Ability to upload and store relevant documents
- Focused conversations within a specific context
- Easy access and organization

### Components

**Instructions:**
- You can add custom instructions specific to the project to guide Claude's behavior within that workspace
- Example: "Act as senior website developer that helps me with tasks related to the redesign of our website"

**File Storage:**
- Upload relevant documents and files
- Provides context for all chats within the project

**Chat History:**
- All conversations are saved and organized within the project
- Easy to review and build upon previous discussions

### Use Cases

Use projects for virtually everything, organizing by:
- Personal tasks (shopping, travel, healthcare research)
- Work-related projects
- Team collaboration tasks
- Self-development goals


## Claude Cowork

Cowork is a downloadable desktop application that allows Claude to interact with your actual computer files and folders, moving AI assistance beyond simple prompt-and-response interactions in a web browser.

### Key Capabilities
- Access, analyze, create/edit files and folders on your computer
- Perform file-based research and writing
- Automate repetitive tasks
- Provide productivity suggestions

### Considerations

Before using it, make sure to:
- Verify it aligns with your company's IT policies
- Consider comfort level with AI accessing your computer files
- Ensure you have necessary permissions

### Future Outlook

For now, not everyone currently feels comfortable with local AI installation.

But within a few years, having Claude Cowork on your computer will be as standard as having Excel, it will become an essential productivity tool.


## Settings

### General

**Personal Preferences**: Allows you to provide specific instructions about your role, expertise, and preferences so Claude tailors all future responses to your needs.


### Memory

Memory-related settings in Claude's Capabilities section:
1. **Search and reference chats**: Allows Claude to search for relevant details from your previous conversations
2. **Generate memory from chat history**: Allows Claude to automatically remember relevant context from your chats. Applies to both regular chats and projects
3. **Import memory from other AI providers**: Transfer memory/context from other AI tools (ChatGPT, Gemini, etc.) to Claude

:::warning
Past conversations from unrelated topics could bias Claude's responses. It's recommended to keep these settings disabled. 

Store contextual information in Projects rather than relying on global memory features, ensuring Claude provides focused, relevant answers without interference from unrelated past conversations.
:::
