# Claude

## Models

The simple rule:
1. Start with **Sonnet**.
2. If the answer feels shallow or misses nuance, run it again on **Opus**. **Most people won't need Opus for everyday work.**
3. Use **Haiku** when you just need something quick and don't need perfection.

### Opus 4.6

**The Deep Thinker**. Maximum brainpower. For when getting it right matters more than getting it fast.

| Metric | Value |
|--------|-------|
| **Speed** | 2/5 |
| **Intelligence** | 5/5 |
| **API cost** | \$15 in / \$75 out per 1M tokens |
| **Context** | 1M tokens |
| **Max output** | 128K tokens max |
| **Extended thinking** | Yes (adaptive) |
| **Agent teams** | Yes |
| **Free plan** | No |
| **Pro plan** | Yes (switch to it) |

Use This Model When:
- Board-level strategy and analysis
- PhD-level science and research
- Multi-agent coordination (Claude Code)
- Large codebase refactoring (10K+ lines)
- Complex financial modeling
- Anything where nuance really matters

**Cost example:** ~$0.90 for a 10K-word research analysis.


### Sonnet 4.6

**Your Daily Driver**. Best balance of smart, fast, and affordable. This is the one you'll use 80% of the time.

| Metric | Value |
|--------|-------|
| **Speed** | 4/5 |
| **Intelligence** | 4/5 |
| **API cost** | \$3 in / \$15 out per 1M tokens |
| **Context** | 1M tokens (beta) |
| **Max output** | 64K tokens max |
| **Extended thinking** | Yes (adaptive) |
| **Agent teams** | No |
| **Free plan** | Yes (default model) |
| **Pro plan** | Yes (default model) |

Use This Model When:
- Writing emails, reports, proposals
- Everyday coding and debugging
- Spreadsheet and data analysis
- Content creation and editing
- Building artifacts and visualizations
- Basically... everything else

**Cost example:** ~$0.18 for a 10K-word report (5x cheaper than Opus)


### Haiku 4.5

**The Speed Demon**. Shockingly capable for how fast and cheap it is. Don't underestimate this one.

| Metric | Value |
|--------|-------|
| **Speed** | 5/5 |
| **Intelligence** | 3/5 |
| **API cost** | \$0.80 in / \$4 out per 1M tokens |
| **Context** | 200K tokens |
| **Max output** | 8K tokens max |
| **Extended thinking** | No |
| **Agent teams** | No |
| **Free plan** | No |
| **Pro plan** | Yes (switch to it) |

Use This Model When:
- Quick questions and lookups
- Simple formatting and rewording
- Sorting and classifying data
- High-volume batch processing
- First drafts you'll edit anyway
- Anything where speed beats depth

**Cost example:** ~$0.05 for a 10K-word classification task (18x cheaper)
