# Prompt Engineering

## What is a Prompt?

A **prompt** is the input you give to an AI model to produce an output.

Think of it as a guide that helps the model understand the context, process information, and generate a relevant and meaningful response.

## Components

Four Components of a Prompt

### Instruction
- Tells the AI model what task to perform
- Examples: summarization, translation, classification
- Sets the stage for the AI's entire response

### Context
- Provides additional information to help the AI better understand the task
- Improves accuracy and performance
- May or may not be necessary depending on the task

### Input Data
- The actual information the AI will process
- Can be text, images, or any relevant data
- What the model works with to complete the assigned task

### Output Indicator
- Signals to the AI model that a response is expected
- Can be **implicit** (built into the instruction) or **explicit** (clearly stated)
- Tells the model when to generate its output
