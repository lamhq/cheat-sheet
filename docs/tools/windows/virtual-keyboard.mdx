# Virtual Keyboard

## Use cases

- Your laptop or a mini-keyboard does not have numpad


## Solution

The on-screen keyboard in Windows 10 and Windows 11 has a virtual numpad that you can use.

To access the on-screen keyboard, use the shortcut key `Windows + Ctrl + O`.
