# Customization

## Introduction

You can customize the behavior of GitHub Copilot in Visual Studio Code, such as:

- Customize chat responses to match your coding practices and project requirements.
- Set up persistent configurations that automatically apply your preferred context, tools, and guidelines to every conversation.

Main ways to customize GitHub Copilot's behavior:
- Custom Instructions
- Prompt Files
- Custom Agents
- MCP and tools
- Language models


## Use Cases

1. Project-wide coding standards: Custom instructions
2. Language or framework-specific rules: Custom instructions with glob patterns
3. Reusable development tasks: Prompt files
4. Use chat for planning or research: Custom agents
5. Define specialized workflows: Custom agents
6. Integrate external services: MCP and tools
7. Bring your own model: Language models
