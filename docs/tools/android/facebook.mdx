# Facebook

## Version

This guide is for Facebook Android app version `470.0.0.61.82`


## View app version

Go to `Settings & privacy`, scroll to the bottom setting `About`.


## Turn of Video Autoplay

Go to `Settings & privacy`, under `Preferences`, select `Media`:
- Check `Never Autoplay Videos`.
- Turn off `Videos start with sound`


## Open links in browser

Go to `Settings & privacy`, under `Preferences`, select `Media`:
- Check `Open links in external browser`
