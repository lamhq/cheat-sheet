# Morphe YouTube

## Download

- Download & install [Morphe Manager](https://vanced.to/Morphe-manager)
- Open Morphe Manager, install `Morphe YouTube`
- Homepage: https://vanced.to/


## Remember playback speed

`Settings` / `Morphe` / `Video`, select `Remember playback speed`


## Set custom playback speeds

`Settings` / `Morphe` / `Video`, select `Custom playback speeds`


## Set default playback speed

`Settings` / `Morphe` / `Video`, select `Default playback speed`


## Set default video quality

`Settings` / `Morphe` / `Video`, select `Default video quality on Wi-Fi network`


## Disable autoplay videos in feed

`Settings` / `General`, set `Playback in feeds` to `Off`.


## Disable video playback on top of other apps

`Settings` / `General`, turn off `Picture-in-picture`


## Adjust volume and brightness by swiping

`Settings` / `Morphe` / `Swipe controls`, turn on:
- **Enable brightness gesture**
- **Enable volume gesture**
