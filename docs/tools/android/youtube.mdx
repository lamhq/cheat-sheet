# Youtube ReVanced

## Download

- Homepage: https://vanced.to/
- Download & install [ReVanced Manager](https://github.com/vancedapps/vanced.to/releases/download/2025-51/vanced.to_revanced_manager_plus_v3.0.22.apk)
- Open ReVanced Manager, install `YouTube ReVanced`


## Set custom playback speeds

`Settings` / `Revanced` / `Video`, select `Custom playback speeds`


## Disable autoplay videos in feed

`Settings` / `General`, set `Playback in feeds` to `Off`.


## Disable video playback on top of other apps

`Settings` / `General`, turn off `Picture-in-picture`


## Ajust volume and brightness by swiping

`Settings` / `Revanced` / `Swipe controls`, turn on:
- **Enable brightness gesture**
- **Enable volume gesture**
