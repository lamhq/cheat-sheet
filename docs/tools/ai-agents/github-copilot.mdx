# GitHub Copilot

## Introduction

GitHub Copilot is an AI coding assistant that helps you write code faster, allowing you to focus on problem solving and collaboration.

This guide is for individual users with GitHub Copilot Pro plan, using Visual Studio Code and GitHub web interface.


## Features

- **Planning**:
   - Copilot Chat can help you brainstorm and identify the best technologies for your project.
   - Copilot Chat can create issues to help track your ideas.
   - Copilot Spaces direct Copilot's responses in the right context for a specific task
- **Scaffolding**
   - Use **GitHub Spark** (a no-code environment) to generate working apps with integrated LLMs.
- **Coding**:
   - Copilot inline suggestions help add code as you type.
   - Next edit suggestions (public preview) predicts the next edit you are likely to make and suggests a completion for it.
   - Copilot Chat can answer questions and offer suggestions in a conversational environment.
   - Copilot coding agent can make code changes, create a pull request for you to review.
- **Reviewing**:
   - Copilot code review gives you feedback in your favorite IDE, or as a pull request review on GitHub.
- **Testing**:
   - Copilot Chat can write, run tests, identify failing tests and fix them (with **MCP Server**).
- **Deployment**:
   - Copilot Chat can help you configure continuous integration and continuous deployment (CI/CD) pipelines.
- **Operation**:
   - You can assign Copilot coding agent to an open issue and it will automatically raise a pull request to address the necessary changes.
   - Copilot Chat can help with tasks you're working on yourself.
   - Copilot pull request summaries generate summaries of a pull request.
- **Security**
   - When code scanning finds a vulnerability, **Copilot Autofix** can be requested to automatically suggest a fix, which a developer then reviews and approves.


## Availability

Copilot is available:

- On the **GitHub website** via Chat
- On the **command line**, through the GitHub CLI
- In your **IDE**
- In **GitHub Mobile**, as a chat interface
- ...


## Getting access

There are few ways you can start using Copilot:

1. Use Copilot Free
2. Sign up for a paid plan:
   - Copilot Pro with a one-time 30-day trial
   - Copilot Pro+
3. Request access to [Copilot](https://github.com/settings/copilot) through your organization (if you are a member of an organization or enterprise that has a subscription to GitHub Copilot)

See [Get access to GitHub Copilot](https://code.visualstudio.com/docs/copilot/setup#_get-access-to-github-copilot).


## Plans

### Copilot Free

- **Audience**: Individual developers not affiliated with an organization or enterprise.
- **Cost**: Free.
- **Features**:
  - Basic access to Copilot code suggestions.
  - Limited usage: 2,000 code completions and 50 premium requests per month.
- **Best for**: Trying out Copilot with minimal commitment.

### Copilot Pro

- **Audience**: Individual developers seeking more power and flexibility.
- **Cost**: $10/month or $100/year. Purchase additional premium requests at $0.04/request.
- **Features**:
  - Unlimited code completions.
  - Access to premium AI models.
  - 300 premium requests per month.
  - Includes the Copilot coding agent.
- **Special Access**: Free for verified students, teachers, and maintainers of popular open-source projects.
- **Best for**: Regular users who want full access to Copilot's capabilities.

### Copilot Pro+

- **Audience**: Power users and professionals needing maximum AI support.
- **Cost**: $39/month or $390/year.
- **Features**:
  - Everything in Copilot Pro.
  - 1,500 premium requests per month.
  - Full access to all available models in Copilot Chat.
- **Best for**: Heavy users who rely on advanced AI features and higher usage limits.

For all plans, check [GitHub Copilot Plans](https://docs.github.com/en/copilot/get-started/plans).
