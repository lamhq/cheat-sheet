# Getting started

## Set up Copilot in VS Code

Prerequisites:
- A GitHub account
- [Access to GitHub Copilot](../github-copilot.mdx#getting-access) with your GitHub account

Follow the [set up Copilot in VS Code](https://code.visualstudio.com/docs/copilot/setup#_set-up-copilot-in-vs-code) guide.


## Use a different GitHub account per workspace or profile

If you use Copilot with different accounts for **work** and **personal** projects, you can sign in to different GitHub accounts for Copilot per VS Code workspace or profile.


## Disable AI feature

- [Remove AI features from VS Code](https://code.visualstudio.com/docs/copilot/setup#_remove-ai-features-from-vs-code)
