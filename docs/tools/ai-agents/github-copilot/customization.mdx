# Customization

## Introduction

You can customize the behavior of GitHub Copilot in Visual Studio Code, such as:

- Customize chat responses to match your coding practices and project requirements.
- Set up persistent configurations that automatically apply your preferred context, tools, and guidelines to every conversation.

Main ways to customize GitHub Copilot's behavior:
- Custom Instructions
- Prompt Files
- Custom Agents
- MCP and tools
- Language models


## Chat Customizations editor

The Chat Customizations editor provides a centralized UI for discovering, creating, and managing all your customizations in one place.

To open the Chat Customizations editor, run **Chat: Open Chat Customizations** from the Command Palette.
