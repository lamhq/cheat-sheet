# GitHub Copilot

## Set up

Prerequisites:
- A **GitHub account**
- A **GitHub Copilot subscription** (Free trial or paid)

Steps:
1. Hover over the Copilot icon in the Status Bar and select Set up Copilot.
![Set up Copilot](https://code.visualstudio.com/assets/docs/copilot/setup/setup-copilot-status-bar.png)
2. Sign in to GitHub in Visual Studio Code
![Sign in to GitHub](https://code.visualstudio.com/assets/docs/copilot/setup/setup-copilot-sign-in.png)

References:
- [Setting up GitHub Copilot for yourself](https://docs.github.com/en/copilot/how-tos/set-up/set-up-for-self)
