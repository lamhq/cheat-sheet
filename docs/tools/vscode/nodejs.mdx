# Node.js

## Running NPM Scripts

Run NPM script without using command line.

1. Open the **Explorer sidebar**.
2. click the **…** (More Actions) menu in the Explorer and check **NPM Scripts** to enable it.
3. Scroll down until you see **NPM Scripts** section.
4. VS Code automatically detects scripts defined in your `package.json`.
5. Click the **play button ▶️** next to the script you want to run.
