## Font setting

To change font:

1. Open an ebook in E-book viewer
2. Go to Settings `Cmd` + `,`
3. Choose `Fonts`

To increase font size:
1. Open an ebook in E-book viewer
2. Right click to the page
3. Select `Font size`


## Styles setting

Set custom CSS for better view:
1. Go to Settings
2. Choose `Styles` item
3. Paste the following css:

```css
h1, h2, h3, h4, h5, h6 {
  color: #2B5282;
}

[class*="block_"] {
  margin: 0.5em 0 !important;
}

p {
  text-align: justify !important;
}
```


## Translate highlight text

1. Select text, choose `Lookup`
2. In the Lookup panel, choose `Add sources`
3. Click `Add`
4. In the form, set **name** to `Google Translate`
5. Set **url** to `https://translate.google.com/#view=home&op=translate&sl=en&tl=vi&text={word}`

Reference [here](https://www.reddit.com/r/languagelearning/comments/iz71f9/comment/g6h4o4m/?utm_source=share&utm_medium=web2x&context=3).
