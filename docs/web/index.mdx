## Topics

Main topics in web development:

- **Development Environment**: set up & optimize your local development environment
- **Authentication**: OAuth, protected components
- **Routing & Navigation**: render routes, redirect
- **Layout**: share layout between pages
- **UI doc**: Storybook
- **UI framework**: Material UI
- **Styling**: TailwindCSS, CSS-in-JS
- **Asynchronous tasks**: query, mutation
- **Global state management**: Jotai, Redux
- **Form handling**: Server Action, React Hook Form
- **Error handling**: Error Boundary
- **Component test**: React Testing Library
- **End-to-end test**: Playwright
- **Code Organization**: Module-based, Atomic Design, Container and Presentational Components
- **Internationalization**: multiple languages
- **Security**: Content Security Policy (CSP), implement security mechanisms
- **Debugging**: Debug application locally in IDE, Browser
- **Instrumentation**: integrate monitoring and logging tools, OpenTelemetry
- **Profilling**: measure application's performance & find bottlenecks
- **Production**: optimize application for production
- **Optimistic Update**: Tanstack Query, `useOptimistic`
- **Rendering Strategies**: Client-side rendering (Single-page Application), Server-side rendering
- **Architectures**: Monolithic Frontend, Modular Frontend, Microfrontend
- **Lazy Loading**: improve the initial loading performance.
- **Incremental Static Regeneration**: pre-render pages at build time
- **PWA**: build Progressive Web Application
