# Vite

## Version

This guide covers Vite version `7.3.1` (2065).


## Overview

Vite (French word for "quick", pronounced `/vit/`) is a modern build tool that provides a significantly faster and leaner development experience for web projects. It consists of two major parts:

- **Dev Server**: A development server with rich feature enhancements over native ES modules, featuring extremely fast Hot Module Replacement (HMR)
- **Build Command**: A production build command that bundles your code with Rollup, pre-configured to output highly optimized static assets


## Installation

<PackageManagerTabs command="create vite@latest" />


## Running Your Project

```sh
npm run dev
```
 
The server will start at `http://localhost:5173` by default.
