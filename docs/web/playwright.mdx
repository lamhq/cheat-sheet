# Playwright

## Version

This document is for version `1.54.1` (2025).


## Installation

Initialize a new project or adds Playwright to an existing one:

```sh
pnpm create playwright
```

The command will:
- Prompt to choose a language (JavaScript, TypeScript)
- Add a GitHub Actions workflow (for CI)
- Download & install Playwright browsers
- Create a Playwright config file with sample tests


## IDE Integration

Install [Playwright Test for VSCode](https://marketplace.visualstudio.com/items?itemName=ms-playwright.playwright) extension allowing you to run, debug, and generate tests using VS Code UI.
