# Rspress

Lightning Fast Static Site Generator.

## Why Rspress

- Build Performance: fast startup speed, good development experience.
- MDX Support.
- Basic Capabilities of Documentation Site: 
  - internationalization
  - multi-version support
  - full-text search
  - custom document theme
- Extensible through plugin


## Quick start

### Create new project

<PackageManagerTabs command="create rspress@latest" />


### Add Rspress to existing project

<PackageManagerTabs command="install rspress -D" />

Add the following script to `package.json`:

```json title="package.json"
{
  "scripts": {
    "dev": "rspress dev --port 5050",
    "build": "rspress build",
    "preview": "rspress preview --port 5050"
  }
}
```

Create a configuration file:

```js title="rspress.config.ts"
import { defineConfig } from 'rspress/config';

export default defineConfig({
  root: 'docs',
});
```

Create a TypeScript configuration file:

```json title="tsconfig.json"
{
  "compilerOptions": {
    "target": "ES2020",
    "lib": ["DOM", "ES2020"],
    "module": "ESNext",
    "jsx": "react-jsx",
    "noEmit": true,
    "strict": true,
    "skipLibCheck": true,
    "isolatedModules": true,
    "resolveJsonModule": true,
    "moduleResolution": "bundler",
    "useDefineForClassFields": true,
    "allowImportingTsExtensions": true
  },
  "include": ["docs", "theme", "rspress.config.ts"],
  "mdx": {
    "checkMdx": true
  }
}
```

### Start dev server

<PackageManagerTabs command="run dev" />


### Build

<PackageManagerTabs command="run build" />

### Preview

```sh
rspress preview --port 8080
```
