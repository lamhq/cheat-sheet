# UI Components

List of commonly used UI components that can be used in documentation pages. 

## PackageManagerTabs

Render install/run commands for multiple package managers in a single tabbed UI.

```md
import { PackageManagerTabs } from '@rspress/core/theme';

<PackageManagerTabs command="install -D @rspress/core" />
```

### Commands per package manager

You can define the command for each package manager by passing an object:

```md
import { PackageManagerTabs } from '@rspress/core/theme';

<PackageManagerTabs
  command={{
    npm: 'npm create rspress@latest',
    yarn: 'yarn create rspress',
    pnpm: 'pnpm create rspress@latest',
    bun: 'bun create rspress@latest',
    deno: 'deno init --npm rspress@latest',
  }}
/>
```

### Add extra tabs

Use `additionalTabs` to include custom package managers or tools:

```md
import { PackageManagerTabs } from '@rspress/core/theme';

<PackageManagerTabs
  command="install -D @rspress/core"
  additionalTabs={[
    { tool: 'custom' }
  ]}
/>
```

### Props

```ts title="types.ts"
type PackageManagerTabProps = (
  | {
      command: string;
      exec?: boolean;
      dlx?: boolean;
    }
  | {
      command: {
        npm?: string;
        yarn?: string;
        pnpm?: string;
        bun?: string;
        deno?: string;
      };
      exec?: never;
      dlx?: never;
    }
) & {
  additionalTabs?: {
    tool: string;
    icon?: React.ReactNode;
  }[];
};
```

- If `command` is a string, the component auto-prefixes the correct package manager command and renders npm, yarn, pnpm, bun, and deno tabs.
- For install commands, yarn/pnpm/bun/deno tabs automatically replace `install` with `add`.
- In the deno tab, packages without explicit source automatically get `npm:` added.


## Prompt

Displays a copyable prompt block for AI instructions with optional folding behavior.

```md
import { Prompt } from '@rspress/core/theme';

<Prompt
  title="Start a Rspress project in one shot"
  description="Copy this prompt and send it to your AI agent. It will scaffold a new Rspress site for you automatically."
  prompt={`Create a Rspress project by following the official quick start guide at https://rspress.rs/guide/start/getting-started.md.

1. Scaffold the project with \`create rspress@latest\`.
2. Install dependencies.
3. Start the dev server.
4. Verify the site renders correctly.`}
/>
```

- The prompt can be folded.
- Click anywhere on the card to copy the full prompt text.

You can put the prompt content in a separate file and import it via `?raw`:

```md
import { Prompt } from '@rspress/core/theme';
import promptText from './_prompt.txt?raw';

<Prompt
  title="Start a Rspress project in one shot"
  description="Copy this prompt and send it to your AI agent."
  prompt={promptText}
/>
```

### Props

```ts title="types.ts"
interface PromptProps extends React.HTMLAttributes<HTMLDivElement> {
  custom?: boolean;
  description?: string;
  title?: string;
  eyebrow?: string;
  defaultCollapsed?: boolean;
  prompt?: string;
}
```

- `title`: header label (`Agent Prompt` by default)
- `eyebrow`: text above the title (`For your Agent` by default)
- `defaultCollapsed`: initial folded state (default `true`)
- `prompt`: prompt text shown and copied


## Tabs

Create tabbed content panes.

```md
import { Tab, Tabs } from '@rspress/core/theme';

<Tabs>
  <Tab label="Tab 1">Tab 1 content</Tab>
  <Tab label="Tab 2">Tab 2 content</Tab>
</Tabs>
```

Code blocks inside tabs:

````md
import { Tab, Tabs } from '@rspress/core/theme';

<Tabs>
  <Tab label="Tab 1">
```tsx
title="src/index.mjs"
import foo from 'foo';
import bar from 'bar';
```
  </Tab>
  <Tab label="Tab 2">
```tsx title="src/index.cjs"
const foo = require('foo');
const bar = require('bar');
```
  </Tab>
</Tabs>
````

### Props

```ts title="types.ts"
interface TabsProps {
  children: React.ReactNode;
  defaultValue?: string;
  groupId?: string;
  tabPosition?: 'left' | 'center';
}

interface TabProps {
  label: string;
  value?: string;
  children: React.ReactNode;
}
```

- `defaultValue`: preselect a tab by matching `value`
- `groupId`: syncs selected tab across multiple `Tabs` groups
- `tabPosition`: aligns the tab list (`left` or `center`)


## Steps

Render step-by-step instruction blocks.

```md
import { Steps } from '@rspress/core/theme';

<Steps>

### Step 1
Content for step 1.

### Step 2
> Content for step 2.

</Steps>
```

- Each `###` heading inside `<Steps>` becomes a new step.
- Content below the heading becomes that step’s body.
