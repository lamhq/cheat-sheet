# Installation

## Using the terminal

Initializes a new project with Playwright, or adds Playwright setup to your current project:

```sh
pnpm create playwright
```

Playwright will download the browsers needed and create:
- Playwright config file
- Sample tests


## Using VS Code

- Install [VS Code extension](https://marketplace.visualstudio.com/items?itemName=ms-playwright.playwright).
- Open the testing sidebar to run tests

For more information, see [Getting started - VS Code](https://playwright.dev/docs/getting-started-vscode)


## Running test

```sh
npx playwright test
```

To run tests with UI Mode:
```sh
npx playwright test --ui
```
