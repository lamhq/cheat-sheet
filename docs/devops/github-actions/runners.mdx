# Runners

## Preinstalled software in GitHub-hosted runners

Workflow logs include a link to the preinstalled tools on the runner.

To find the link:
1. Go to your repository on GitHub.
2. Click on the "Actions" tab.
3. Select a workflow run from the list.
4. Click on a job in the workflow run.
5. Expand the `Set up job` section.
6. Expand the `Runner Image` section.
7. The link following `Included Software` will describe the preinstalled tools.
