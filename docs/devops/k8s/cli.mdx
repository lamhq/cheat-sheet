# CLI commands

## Apply

Create or update objects defined in YAML file:
```sh
kubectl apply -f pod.yml -f deployment.yml
```


## Pod

List all Pods:
```sh
kubectl get pods
```

Describle Pod:
```sh
kubectl describe pod <POD_NAME>
```

Delete Pod:
```sh
kubectl delete pod <POD_NAME>
```

Get container's log:
```sh
kubectl logs <POD_NAME> --container <CONTAINER_NAME>
```

Run commands in Pods:
```sh
kubectl exec <POD_NAME> -- <COMMAND>
```

Get shell access to a container:
```sh
kubectl exec -it <POD_NAME> -- sh
```

## Deployment

List all deployments:
```sh
kubectl get deployments
```

Delete deployment:
```sh
kubectl delete deployment NAME
```

## Node

List all the nodes in cluster:
```sh
kubectl get nodes
```

## Config

View your config:
```sh
kubectl config view
```

## Context

View current context:
```sh
kubectl config current-context
```

List all the contexts defined in your kubeconfig file:
```sh
kubectl config get-contexts
```

Switch the current Kubernetes context:
```sh
kubectl config use-context my-context
```


## Namespaces

List all namespaces:

```sh
kubectl get namespaces
```

Get information about a namespace:

```sh
kubectl describe ns <NAMESPACE>
```

List all Service objects in namespace:

```sh
kubectl get svc --namespace kube-system
```

Switch to a namespace:

```sh
kubectl config set-context --current --namespace <NAMESPACE>
```

Create a new Namespace:
```sh
kubectl create ns hydra
```
