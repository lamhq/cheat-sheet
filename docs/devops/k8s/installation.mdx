# Installation

Installation instruction for macOS

## Using Docker Desktop

The simplest way to install Kubernetes on a local machine is by using Docker Desktop.

To enable Kubernetes in Docker Desktop:
- Open Docker Desktop.
- Go to **Settings** â†’ **Kubernetes**.
- Check **Enable Kubernetes**.
- Click **Apply & Restart**.


## Verify installation

```sh
kubectl version
kubectl get all
```


## Setting command alias

Add a command alias for `kubectl` on macOS using `zsh`.

Append this line to your `~/.zshrc`:
```sh
alias k=kubectl
```

Reload your shell to apply changes:
```sh
source ~/.zshrc
```
