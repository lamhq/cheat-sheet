# Turborepo

## Introduction

Turborepo is a high-performance build system for JavaScript and TypeScript codebases.

It is designed for scaling monorepos.


## Key Features

- **Content-aware caching** – stores build outputs keyed by file content, so unchanged code is instantly reused.  
- **Parallel execution** – runs tasks across packages simultaneously, maximizing CPU usage and cutting build times.  
- **Remote caching** – lets teams share cached results across machines, avoiding duplicate work in CI/CD pipelines.  
- **Incremental builds** – rebuilds only the packages affected by recent changes, keeping feedback loops tight.  
- **Monorepo scaling** – efficiently manages thousands of tasks across large multi-package repositories.  
- **Custom pipelines** – supports declarative task pipelines for linting, testing, and building, tailored to project needs.
