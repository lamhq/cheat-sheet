# Todos

The `todos/` directory contains a list of approved development tasks ready for implementation, ordered from oldest to newest.

Tasks can be imported from external tools (Jira, Trello, etc.) or manually maintained.

Each task is documented in a separate markdown file.

## Task Types

| **Type** | **File name format** | **Example** |
|----------|------------|-------------|
| New Feature Request | `{task-id}-feat-<feature-name>.mdx` | `001-feat-login.mdx` |
| Change Request | `{task-id}-crq-<change-name>.mdx` | `002-crq-disable-email-login.mdx` |
| Bug Report | `{task-id}-bug-<bug-name>.mdx` | `003-bug-login-special-characters.mdx` |


## Task Status

- **Pending** — Approved, ready to implement
- **In Progress** — Being worked on
- **Done** — Completed with links to issues, PRs, and commits
