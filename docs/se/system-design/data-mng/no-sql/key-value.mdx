# Key-Value Database

## Overview

Stores data as a collection of key-value pairs.

Each item has a unique key and a value (can be primitive values or complex objects).

Extremely fast for lookups by key.

Highly scalable and performant for simple read/write operations.

Use cases:
- Caching
- User session managements
- Real-time data lookups


## Key/Value Database Solutions

- Redis
- Aerospike
- Amazon DynamoDB
